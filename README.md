# 신고가 레이더

코스피·코스닥에서 52주 최고가에 가까운 종목을 찾는 로컬 대시보드입니다.

## 실행

`실행하기.ps1`을 우클릭해 PowerShell로 실행하거나, 이 폴더에서 `node server.mjs`를 실행한 뒤 <http://127.0.0.1:4173>을 여세요.

실행 전에 `.env.example`을 참고해 `APP_PASSWORD`와 `SESSION_SECRET` 환경 변수를 설정하세요. 공개 웹 배포에서는 비밀번호를 코드나 GitHub에 넣지 말고 배포 서비스의 비밀 환경 변수로 등록해야 합니다.

## 웹 배포

`render.yaml`을 이용해 Render에 배포할 수 있습니다. 배포 설정에서 `APP_PASSWORD`를 지정하면 접속 시 비밀번호 로그인 화면이 표시됩니다. `SESSION_SECRET`은 자동으로 생성됩니다.

감시 PC에서는 `서버자동실행.ps1`이 로컬 서버와 Cloudflare 임시 외부 링크를 함께 실행합니다. 생성된 주소는 `외부링크.log`의 `trycloudflare.com` 주소입니다.

대시보드의 **오늘 데이터 갱신**을 누르면 네이버 금융 공개 시세에서 최신 목록을 받습니다. 첫 갱신은 후보 수와 네트워크 상태에 따라 수 분 걸릴 수 있습니다. 갱신이 끝나기 전까지 이전 데이터는 그대로 유지됩니다.

코스피·코스닥 일반주식 전 종목을 수집하고 화면에서 조건을 적용합니다. 기본 조건은 `52주 최고가 대비 -20% 이상`, `시가총액 2,000억원 이상`, `1봉전(전 거래일) 거래량 150,000주 이상`입니다. 조건 변경, 종목 검색, 정렬, CSV 저장을 지원합니다.

> 네이버 금융의 비공식 공개 API를 사용합니다. 정보 제공용이며 자동 주문 기능은 없습니다.

## 매일 오전 7시 텔레그램
